# Depth First Search (DFS)

## Program Description
This program does a Depth First Traversal (DFS) on a graph and outputs the nodes in the order they are visited.

## Inputs
- Number of vertices
- Number of edges
- Pairs of connected nodes (edges)
- Starting node

## Outputs
- Order of nodes visited using DFS traversal

## How to Compile and Run

### Compile:
g++ main.cpp -o dfs

### Run:
./dfs
